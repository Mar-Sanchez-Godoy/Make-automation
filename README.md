# Make-automation
This automation, built in Make, connects Gmail, Google Sheets, and AI (OpenAI, Gemini, Mistral) to create an intelligent workflow that processes incoming emails, generates AI‑powered content, and responds automatically.

## ⚙️ Automation with Make: Gmail → Google Sheets → AI → Gmail Integration  
This automation, built in Make, connects Gmail, Google Sheets, and OpenAI to create an intelligent workflow that processes incoming emails, generates AI‑powered content, and replies automatically. The goal is to reduce manual tasks and centralize information in a spreadsheet.

## 🔄 Flow Description  
Gmail – Watch Emails  
The scenario is triggered when a new email arrives in the inbox. Make automatically captures the sender, subject, and message content.

Google Sheets – Add a Row  
The email information is logged into a Google Sheets file, creating a structured record for future analysis.

AI– Generate Completion  
The email content is sent to an OpenAI model to generate a summary, classification, suggested reply, or any defined processing (e.g., intent analysis, sentiment detection, or key data extraction).

Google Sheets – Update a Row  
The previously created row is updated with the AI‑generated output, keeping the spreadsheet as the single source of truth.

Gmail – Send Email  
Finally, Make sends an automatic reply to the sender, using the text generated by OpenAI or a custom template.

## 🧩 Added Value of This Automation

Reduces response times.

Centralizes data in Sheets for BI analysis.

Enables AI‑enhanced email processing without manual intervention.

Scalable: validations, filters, branches, or additional integrations can be added.

# Castellano

# Automatización en Make.com

## ⚙️ Automatización con Make: Integración Gmail → Google Sheets → OpenAI → Gmail
Esta automatización, desarrollada en Make, conecta Gmail, Google Sheets y OpenAI para crear un flujo inteligente que procesa correos entrantes, genera contenido con IA y responde automáticamente. El objetivo es reducir tareas manuales y centralizar la información en una hoja de cálculo.

## 🔄 Descripción del flujo

Gmail – Revisar mails
El escenario se activa cuando llega un nuevo correo a la bandeja de entrada. Make captura automáticamente el remitente, asunto y contenido.

Google Sheets – Añadir a una base de datos
La información del correo se registra en una hoja de Google Sheets, creando un historial estructurado para análisis posteriores.

IA - Generar una respuesta 
El contenido del correo se envía al modelo de OpenAI para generar un resumen, clasificación, respuesta sugerida o cualquier procesamiento definido (por ejemplo: análisis de intención, sentimiento o extracción de datos clave).

Google Sheets – Añadir respuestas a la base  
La fila previamente creada se actualiza con la salida generada por la IA, manteniendo la hoja como fuente única de verdad.

Gmail – Enviar respuesta por mail
Finalmente, Make envía una respuesta automática al remitente, utilizando el texto generado por OpenAI o una plantilla personalizada.

## 🧩 Valor añadido de esta automatización
Reduce tiempos de respuesta.

Centraliza datos en Sheets para análisis BI.

Permite enriquecer correos con IA sin intervención manual.

Escalable: se pueden añadir validaciones, filtros, ramificaciones o integraciones adicionales.

## Conclusiones

Una automatización ligera pero potente que convierte correos cotidianos en información estructurada y enriquecida con IA — totalmente escalable y sin intervención manual.
